# 🏍️ Sistema de Ventas de Motocicletas

Sistema web profesional para el registro, análisis y visualización de ventas de motocicletas.

## 🚀 Instalación

1. Sube todos los archivos a GitHub
2. Configura Google Apps Script (ver appscript/Code.gs)
3. Actualiza la URL en js/config.js
4. Despliega en Vercel

## 📊 Características

- Dashboard ejecutivo con KPIs
- Captura de ventas con validaciones
- Análisis avanzado con filtros
- Exportación a CSV
- Diseño responsive y dark mode

## 🛠️ Tecnologías

- HTML5, CSS3, JavaScript Vanilla
- Chart.js para gráficas
- Google Sheets + Apps Script como backend
